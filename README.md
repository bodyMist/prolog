## Convention
### naming
* Camel Case
* Constant : Underscore(_), Capital
### indent
* 4 spaces
### line break
* 쉼표(,) 이후에 줄 나누기
* 줄 나눌 때 연산자 앞에서 나누기
* 앞 줄과 같은 레벨의 새로운 줄은 들여쓰기를 일치시키기
### comment (주석)
* 짧은 단문은 해당 코드의 우측 편 1TAB
* 메서드의 중요 로직 또는 추가 설명이 필요한 경우는 해당 코드의 상단에서 블럭주석으로 작성
* 클래스 및 메서드에 대한 문서화 주석은 선언문 위에서 작성
### Bracket
* 중괄호는 내려쓰지 말기
### Declaration
* 한 줄에 하나 또는 동일 타입 다수 선언
* 한 줄에 여러 타입 선언 X
* 스페이스로 들여쓰기 맞추기 X
